AltFire:
BLZB B 0 A_JumpIfNoAmmo("NoAmmo")
BLZB BC 2
BLZB D 0 A_PlaySoundEx("weapon/chargekick", "Weapon") 
BLZB D 0 A_FireCustomMissile("KnifeSlash",38,0,0,0)

Botfire:

BLZC A 35

Goto Ready+1

NoAmmo:

}
}
